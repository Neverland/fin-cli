
# fin-cli

 fin-cli  曼哈顿cli

## start
```javascript
    npm install fin-cli -g

```

## usage

### fin init

#### option

    -uri|u git仓库所在的uri

```javascript
    fin install -u git.fin.com
```

### fin create

#### option

    -type|t 类型
        1. type: component、
        2. type: page
 
```javascript
    fin create -t component
```

### fin config

#### option

    -email|e email前缀
    -author|a auhtor
 
```javascript
    fin config -e abc
    fin config -a abc
    fin config -e abc -a abc
```
